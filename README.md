### Hooks
- `pre-commit` hook to format with prettier.
- `pre-push` hook to run tests (far future).
- configure with git
```
git config core.hooksPath hooks/*
```
- give the files execute perms
```
chmod +x hooks/*
```
- install prettier
```
npm install --save-dev --save-exact prettier
```
